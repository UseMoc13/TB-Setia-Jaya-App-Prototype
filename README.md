# TB Setia Jaya App Prototype

## Project Overview
This project was developed during the second semester at UMN
The application is based on the business model of **Setia Jaya Building Supply Store**.

The outcome of this project includes:
- A UI/UX prototype design
- A database design for the **TB Setia Jaya App**

The application was designed using Figma

---

## Prototype Flow
The prototype consists of three main flows:

### 1. Main Flow
Displays the opening screen of the application and directs users to the login menu.

### 2. Admin Login Flow
Displays the main menu and features available for admin users.

### 3. Employee Login Flow
Displays the main menu and features available for employee users.

---

## Tools & Technologies
- Figma
- SSMS
- Microsoft MYSQL

---

## Project Purpose
The purpose of this project is to create a digital solution for managing building supply store operations through a modern and user-friendly application prototype.
