## 🎨 Prototype Flow
The prototype consists of three main flows:

### 1. Main Flow
Displays the opening screen of the application and directs users to the login menu.

### 2. Admin Login Flow
Displays the main menu and features available for admin users.

### 3. Employee Login Flow
Displays the main menu and features available for employee users.
